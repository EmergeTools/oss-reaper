Reaper API Endpoints

  • POST /report - Endpoint for Reaper SDK to submit class usage reports
  • POST /reaper/error - Endpoint for Reaper SDK to submit error reports
  • GET /analyze - View statistics about collected reports

Setting up Reaper with this backend

Add the following to your Android manifest:

<meta-data
  android:name="com.emergetools.OVERRIDE_BASE_URL"
  android:value="http://your-server-url" />
